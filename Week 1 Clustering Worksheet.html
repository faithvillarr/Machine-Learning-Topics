K-Means Clustering

Clustering is a method of unsupervised machine learning. We are analyzing unlabeled data and organizing it through our model.

K-Means Clustering is a simple unsupervised model used to identify clusters of similar data. The programmer chooses the number of clusters kkk﻿ to be made.

The number of clusters kkk﻿ determines the number of centroids c(1),...,c(k)c^{(1)},...,c^{(k)}c(1),...,c(k)﻿. These centroids can be randomly initialized to some location in our feature space. Points can then be assigned to a centroid/cluster by calculating the smallest Euclidean distance between itself and one of the centroids.

Lloyd’s Algorithm

Lloyd followed the idea of random initialization and taking incrementing the function in the two steps listed above.

First, for every point, we assess what centroid is nearest and reassign points to centroids where necessary.

Next, update each centroid to be located in the mean position of all the points in their cluster.

Greedy Initialization - K-means++ Algorithm ⚠️

This algorithm is used to have slightly better initialization of the centroids.

First, we select a random data point and set that as our first centroid.

Then, we select the farthest point from our first centroid. This is set as our second centroid.

Continue to find the next farthest point from each centroid, until you have initialized your clusters.

Finally you can proceed to the standard E-M algorithm to find a local minimum.
